What is an API?

One of the most important things, when it comes to front end development is the ability to get data from a server/service. How you do that is by using an API.

Components Of API

The first thing when it comes to using API's is to get an access token. This token allows you to be identified by a server.

What does that mean?

An API is generally run over a server. This server sits in a room and listens for requests. Now if anyone was allowed to send a request, people could easily take advantage of this and make something like this: 


                while(True){
                    makeStupidRequestToAPIService();
                }

This would cost the service providers a lot of money and may even crash their system. So in order to protect from these things, the service providers assign you with an API key. Now when you send a request to their server, they know who sent it and will be able to provide data accordingly.
