What are Vectors?

The Physics Point Of View

When we take a look at physics, we find that Vectors are present everywhere. Displacement, Velocity, Acceleration are the very basic depiction of Vectors. Lets take a look at what they are referring to.

Foundation:

The premise of a Vector is that its a collection of number that can be plotted on a graph. Though thats not the best way to explain a vector, its more concise to establish an understanding. Upon breaking a vector down you get what are called Components. These components are values on a Cartesian Axis. So the ideal 2D plain contains an x value and a y value. A point that is contained within this plain is represented by : <x y>. Where x and y can be all sorts of numbers. A 3D space will be represented like this: <x y z>, and a 4D space will be represented like this: <x y z w>. Honestly you can use what ever name for the axis you want. These values are the components. Vector Multiplication

There are 3 different kinds of Vector Multiplication:

- Scalar
- Dot
- Cross

Scalar

This is a type of multiplication where you take a numerical input and multiply it to the vectors components, this generally leads to linear scaling of the vectors components


            scalar(num){
              this.x *= num;
              this.y *= num;
            }

Dot

A dot product is going to be more difficult to understand. Its a type of multiplication which results in a scalar output, Lets take a look at what to code looks like


            dot(vec){
              return(
                (this.x * vec.x) + (this.y * vec.y)
              );
            }

This product is a very significant concept in linear algebra and physics. In a general case we would want to do a dot product between 2 normalized vectors. To understand what this does, lets take a look at an example.

When 2 vectors are perpendicular

            
            const v1 = new Vector(1, 0);
            const v2 = new Vector(0, 1);
            console.log(v1.dot(v2));  // This will result as 0

When 2 vectors are parallel

            
            const v1 = new Vector(1, 0);
            const v2 = new Vector(1, 0);
            console.log(v1.dot(v2));  // This will result as 1

When 2 vectors are in opposite direction

            
            const v1 = new Vector(1, 0);
            const v2 = new Vector(-1, 0);
            console.log(v1.dot(v2));  // This will result as -1

So what is going on here?

In the first case:
The acos of 0 is 1.5707963267948966 which translates from radians to about 90 degrees.
In te same way when you take a look at the second case.
The acos of 1 is 0 degrees, and in the third case the acos of -1 is 180 degrees.
So the number being spit out from dot products can be used to get angles between 2 vectors. The correct way of saying this would be, A dot products gives us how much a vector gets influenced by another vector.

In a tug of war with teams of 5 people, the influence of your teammate on you will be 1, cumulatively 5, the influence of your opponent on you will be -1 individually and -5 cumulatively. If there was a weight attached to the center, it would not really influence this dynamic. When you take a dot product, you can get a quantity that depicts influence.
